![Image-1](Active Skills\ActiveSkills_1.png)

------------------------------------------------------------------------------------------------------------  
[Pals](https://palworld.wiki.gg/wiki/Pals) can use Active Skills to fight against their foes.   
## Using Active Skills  
Pals naturally learn their own set of Active Skills as they level up.  
You may use a [Skill Fruit](https://palworld.wiki.gg/wiki/Skill_Fruits) to teach any Pal the skill it contains. There is no limit to how many Active Skills a Pal may learn, except that they may only learn each one once.   
Each Pal can have up to 3 Active Skills enabled at a time. You can swap a Pal's learned Active Skills by viewing them via the Party menu or the Palbox.  
When [riding](https://palworld.wiki.gg/wiki/Mounts) a Pal, a <a href="https://palworld.wiki.gg/wiki/Player">player</a> can use the Pal's Active Skills manually.  
Each skill has a cooldown time (CT) before it may be used again.
